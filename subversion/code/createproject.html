Create a new ISIP software project

You use this page to automatically create a new Subversion repository and Trac project management project for any Software development you want to do at the ISIP. This is not only for C/C++ or Java based projects, but you are also welcome to create a project for your collection of matlab scripts. Have fun! -Simon

Unix Folder Name:
(Example: biosignalscripts) Please avoid spaces here.

Project Title:
(Example: Biosignal Analysis Scripts) Please avoid quotation marks " and ' here.

Project Description:

Project Owner: (LDAP name)
(Example: simon) Only the owner and root will be able to change access rights later.

Initial Access Rights:
Example:

  * =
  simon = rw
  kier = r

means:
  everyone (*) has no access (* = )
  except simon, who has full access (simon = rw)
  and kier, who has read access (kier = r).
  (one LDAP name per line)
